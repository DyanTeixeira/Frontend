# 🛒 frontend

Bem-vindo ao meu projeto de Frontend! 👋

Este repositório contém o desenvolvimento de um sistema simples e funcional para o gerenciamento de produtos, criado como parte de um trabalho acadêmico/estágio de frontend.

## 🚀 Sobre o Projeto

O objetivo deste projeto foi criar uma interface intuitiva para o controle de estoque, permitindo que o usuário realize operações de login, visualize, adicione e edite produtos.

### 🛠️ Tecnologias Utilizadas

Para a construção deste projeto, utilizei:
- **HTML5**: Estruturação semântica das páginas.
- **CSS3**: Estilização personalizada, com foco em uma interface limpa e responsiva.
- **JavaScript (Vanilla)**: Lógica de front-end para manipulação do DOM e gerenciamento de modais.
- **FontAwesome**: Ícones modernos para melhor experiência do usuário.
- **Google Fonts**: Tipografia selecionada para legibilidade.

## 📂 Estrutura do Repositório

O projeto está organizado da seguinte forma:
- `home.html`: Dashboard principal com a listagem e ações de produtos.
- `login1.html` & `login2.html`: Diferentes interfaces de autenticação.
- `css/`: Arquivos de estilo (incluindo estilos base e específicos por página).
- `js/`: Scripts para o funcionamento dinâmico da aplicação.
- `img/`: Ativos visuais (logos e ícones).

## 🎮 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/DyanTeixeira/dyantrabalhofrontend.git
   ```
2. Abra o arquivo `login1.html` no seu navegador preferido.
3. Navegue entre as telas de login e a dashboard de produtos.

## 📝 Observações

Este projeto foi desenvolvido com foco em praticar conceitos de manipulação de estados no front-end e estilização avançada com CSS.
